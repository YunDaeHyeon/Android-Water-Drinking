# WaterDrinking - 물 섭취 습관 개선 어플리케이션

# Introduction
: 물병의 무게를 재는 어플리케이션 구축  
기능 1 : 물병의 무게를 클라이언트에 새로고침 버튼을 눌러 확인할 수 있는 기능  
기능 2 : 사용자는 시간을 맞춰 약을 먹을 수 있도록 알림을 설정할 수 있는 기능  
기능 3 : 자신이 먹은 물의 양에 따라 달라지는 이모티콘 3가지 (웃음, 보통, 슬픔)
